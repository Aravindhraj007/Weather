# Weather App

A Simple Weather app built with React.js and Api

### Version

1.0.0

### Built Using

- React.js
- Styled components
- Axios
- Api

## Usage

### Clone

Clone the Project

```
$ git clone https://github.com/Aravindhraj007/weather.git
$ cd weather
```

### Installation

Install the dependencies

```
$ npm install
```

### Run

This will run your dev server at http://localhost:3000

```
$ npm start
```
